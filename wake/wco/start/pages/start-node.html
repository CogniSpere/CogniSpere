How to Start a Node

A WCO node is any space — digital or physical — where entries are logged under the WCO banner. Starting one requires almost nothing:

Minimum Viable Node

  • A single page or file displaying the WCO principles.
  • At least one civic entry (observation, story, or document).

Optional Features

  • Local Issues Log – A running page where town/city concerns are logged.
  • Global Resonance Log – Notes on how local issues echo elsewhere.
  • Participation Roles – Listing Initiators, Mirrors, Connectors, etc.
  • Mirrors – Republishing WCO entries from other nodes.

Formats

Anything works — a personal site, a shared blog, a PDF archive, even a Twitter/X thread.

Tagging Convention

Use a simple reference such as [WCO + location + date] to make your entry visible (e.g., WCO-Nogales-2025-08-27).

Every node is autonomous. There is no approval process. By declaring your node as WCO, you connect it into the commons.
